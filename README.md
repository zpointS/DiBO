# DiBO: Training Diffusion Language Models for Black-Box Optimization

Official repository for the ICML 2026 Spotlight paper:

**Training Diffusion Language Models for Black-Box Optimization**

DiBO adapts diffusion language models to offline black-box optimization through delimiter-based domain adaptation and post-training with supervised fine-tuning and reinforcement learning.
